# Achievements / Proof

## Overview
The team prepared a **Technology for the Humanity** project document associated with TEKNOFEST 2025. This section summarizes the documented work and provides supporting evidence.

## Team and Official Identifiers
- Competition / Track: **Technology for the Humanity** 
- Year: **2025**

## What the Team Built (high-level)
The team described a firefighting-assist robot concept that:
- Operates in hazardous fire environments where humans cannot enter
- Uses sensors and a camera module for environmental awareness
- Reports conditions to a web interface using an ESP32-based system
